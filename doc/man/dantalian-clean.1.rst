dantalian-clean(1) -- Clean broken symlinks
===========================================

SYNOPSIS
--------

**dantalian** **clean** [*options*] [*dir*]

DESCRIPTION
-----------

Remove all broken symlinks.  Use current directory if no path is given.

OPTIONS
-------

-h, --help   Print help information.
