dantalian-tags(1) -- List tags of files
=======================================

SYNOPSIS
--------

**dantalian** **tags** [-h] [--print0] *file*

DESCRIPTION
-----------

This command lists the tags of the given file.

OPTIONS
-------

**-h**, **--help**
    Print help information.

**--print0**
    Print the tags separated with NULLs instead of newlines.
